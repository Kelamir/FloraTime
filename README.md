# FloraTime

A pomodoro app that allows you to grow flowers as you get work done.

## Getting Started
	git clone https://github.com/Kelamir/FloraTime
    cd FloraTime
    npm install
    npm run dev -- --open


Add .env file with the following parameters:

    AUTH0_CLIENT_ID=
    AUTH0_CLIENT_
    AUTH0_DOMAIN=
    AUTH0_AUDIENCE=
